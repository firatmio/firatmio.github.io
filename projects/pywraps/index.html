PyWraps - Python Decorator Koleksiyonu

Python geliştiricileri için güçlü ve kullanışlı decorator koleksiyonu

PyPI'da Görüntüle

Proje Hakkında

PyWraps, Python geliştiricilerinin sıkça ihtiyaç duyduğu işlevleri sağlayan güçlü bir decorator koleksiyonudur. Retry mekanizmaları, timeout kontrolü, debounce işlemleri ve background execution gibi yaygın ihtiyaçları karşılayan bu paket, kod kalitesini artırır ve geliştirme süreçlerini hızlandırır.

Proje PyPI'da yayınlanmış olup, basit bir pip install pywraps komutuyla kullanıma hazırdır. Hem senkron hem de asenkron fonksiyonları destekler.

Ana Özellikler

  • @retry - Başarısız fonksiyon çağrılarını otomatik olarak yeniden dener
  • @timeout - Fonksiyonlar için zaman aşımı kontrolü sağlar
  • @debounce - Hızlı ardışık fonksiyon çağrılarını engeller
  • @background - Fonksiyonları arka plan thread'lerinde çalıştırır
  • Asyncio Desteği - Asenkron fonksiyonlarla tam uyumluluk
  • Kolay Entegrasyon - Minimal kurulum ve kullanım
  • Kapsamlı Dokümantasyon - Detaylı kullanım örnekleri
  • Test Coverage - %90+ test kapsamı
